# Inventory Management App

Ce projet est une application de gestion de stock développée avec HTML, CSS et JavaScript. Il permet de mettre en pratique des compétences front-end et de créer une interface fonctionnelle pour gérer les utilisateurs, les produits et les catégories.

## Fonctionnalités

- Page de connexion avec authentification locale
- Création d'un administrateur par défaut
- Dashboard interactif
- Section inventaire pour afficher et gérer les produits
- Gestion des catégories
- Interface responsive avec barre de recherche

## Structure du projet

- `public/index.html` : page principale de l'application
- `src/css/` : styles CSS
- `src/js/` : logique JavaScript
- `assets/` : images et polices

## Installation / utilisation

1. Cloner le dépôt sur votre machine.
2. Ouvrir le dossier du projet.
3. Ouvrir `public/index.html` dans un navigateur.

> Astuce : utilisez une extension Live Server ou un serveur local pour une meilleure expérience.

- Les données sont gérées localement dans `localStorage`.
